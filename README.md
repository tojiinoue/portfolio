# Toji Inoue Portfolio

Toji Inoue のポートフォリオサイトです。  
技術、AI、ビジネスを横断しながら価値をつくるエンジニアとしてのプロフィール、経験、個人開発、発信内容を1ページにまとめています。

## Overview

このサイトは、単なる実績一覧ではなく、以下を一貫したトーンで伝えることを目的にしています。

- どんな視点で開発しているか
- どんな技術領域に関心があるか
- どんなプロダクトや発信をしているか

デザインテーマは `Terminal Noir`。  
ターミナルやコードエディタの空気感をベースに、ダークトーンと高コントラストの配色で「実際に手を動かすエンジニア」の印象を前面に出しています。

## Contents

- Hero: 一言で伝えるプロフィールとスタンス
- About: 自己紹介と価値観
- Skills: 扱っている技術領域
- Experience: これまでの経験
- Projects: 個人開発プロダクトの紹介
- Articles: Zenn での発信内容
- Contact: GitHub、Zenn、プロダクト、メールへの導線

## Featured

掲載している個人開発プロジェクト `幹事くん` は、  
`cc-sdd × OpenAI Codex` を活用した仕様駆動開発で立ち上げた飲み会管理Webアプリです。

ポートフォリオ全体でも、以下のテーマを強く打ち出しています。

- AI と協調しながら実装速度を上げること
- 仕様駆動で曖昧さを減らし、開発を前に進めること
- 作るだけで終わらず、発信まで含めて価値にすること
